## About Me
🎓 Computer Science @ Cal Poly SLO (4th Year)  
📊 Statistics Minor  

**Interests:** Finding patterns in messy data, statistical analysis, and turning numbers into insight. Some cool projects include a deepfake detection classifier with ResNet, student dropout prediction using XGBoost, and my senior project involves simulating satellite cyberattacks to detect false data injection using Extended Kalman Filtering.

🔭 Currently researching explainable AI methods for clinical decision support, focused on electroencephalogram (EEG) time series analysis and seizure classification.

🎣 When I'm not coding, you'll find me fishing along the California coast.
